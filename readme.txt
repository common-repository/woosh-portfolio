== Description ==

Simply display your design projects and descriptive text. This plugin can display 4 images from the Media Library, a YouTube video or a Vimeo video. You can view a demo of the plugin on my [Portfolio Site](http://jacobsommerfeld.net/woosh-portfolio)


== Installation ==

1. Place in wp-content/plugin
2. Type in a name for a portfolio then click "New Portfolio"
3. Select media type then click "New section"

* Media Library
1. Click on add image to select an image
2. Enter data for header, view content url, and text sections
3. Click Save

* YouTube
1. Go to your YouTube Video
2. Copy the part of the url after "v="
3. Paste it into the "View Content" field
4. Add header and text then click Save.

* Vimeo
1. Go to your Vimeo video
2. Click share...it looks like a paper airplain
3. Copy the link and paste into View Content URL
4. Remove all the url except the number at the end

4. Add shortcode [woosh name="name"] to a blank page or post.
